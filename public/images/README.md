# Images Directory

このディレクトリには、トップページで使用する画像を配置します。

## 必要な画像

- `hero.jpg` - ヒーローセクション用（やんばるの森の雰囲気）
- `farm.jpg` - 農園カード用（農園・緑の雰囲気）
- `beans.jpg` - 豆カード用（コーヒー豆の接写）

## 現在の状態

現在はUnsplashの画像URLを直接使用しています。必要に応じて、実際の画像ファイルをこのディレクトリに配置して使用できます。
